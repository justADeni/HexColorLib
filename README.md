# HexColorLib
Simple library for using hex color codes in spigot messages

to add to your project:

<myxml>
<repositories>
		<repository>
		    <id>jitpack.io</id>
		    <url>https://jitpack.io</url>
		</repository>
	</repositories>
  
  <dependency>
	    <groupId>com.github.justADeni</groupId>
	    <artifactId>HexColorLib</artifactId>
	    <version>Tag</version>
	</dependency>
  </myxml>
